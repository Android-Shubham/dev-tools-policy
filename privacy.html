Privacy Policy

This Privacy Policy explains how the Dev Tools application handles information when you use the app.

Dev Tools is a collection of developer utilities including JSON formatting, regex testing, encoding tools, and API testing features.

Information Collection

The app does not collect personal information such as name, email address, phone number, or location.

All developer tools operate locally on your device.

Analytics

The app uses Firebase Analytics to understand how users interact with features. This helps improve the developer tools and prioritize useful functionality.

Analytics may collect anonymous usage information such as:

  • Which tools are opened
  • How often tools are used
  • General app interaction events

This data is anonymous and cannot identify individual users.

Crash Reporting

The app uses Firebase Crashlytics to collect crash reports. This allows the developer to identify and fix stability issues.

Crash reports may include device type, Android version, and technical crash logs.

Performance Monitoring

The app uses Firebase Performance Monitoring to measure application performance such as network request latency and app startup time.

Data Storage

All tool inputs (such as JSON text or API requests) remain on the device and are not transmitted to external servers except when the user explicitly sends an API request through the API testing tool.

Third-Party Services

This application uses the following third-party services:

  • Firebase Analytics
  • Firebase Crashlytics
  • Firebase Performance Monitoring

These services may collect anonymous technical information in accordance with their own privacy policies.

Children’s Privacy

This application is not directed toward children under 13. The app does not knowingly collect personal information from children.

Changes to this Privacy Policy

This Privacy Policy may be updated periodically. Updates will be reflected on this page.
